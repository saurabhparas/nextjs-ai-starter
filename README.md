# Next.js AI Starter

A production-ready Next.js template with AI capabilities built-in.

## Features

- OpenAI Integration - Chat completions, embeddings, and function calling
- Supabase Auth - Authentication with magic links and OAuth
- Tailwind CSS - Utility-first styling with dark mode  
- AI Dashboard - Monitor API usage and costs
- Streaming - Real-time AI response streaming

## Quick Start

```bash
npx create-next-app -e https://github.com/saurabhparas/nextjs-ai-starter
cd nextjs-ai-starter
cp .env.example .env.local
npm run dev
```

## Environment Variables

Create a `.env.local` file with your API keys.

## AI IDE Configuration

This project includes AI coding assistant configurations:
- `.cursorrules` - Cursor IDE rules
- `CLAUDE.md` - Claude Code project context
- `.github/copilot-instructions.md` - GitHub Copilot instructions
- `.windsurfrules` - Windsurf IDE rules
